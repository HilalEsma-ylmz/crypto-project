=================================================================
  WIRESHARK'TA WEBSOCKET MESAJLARI NASIL GÖRÜLÜR
=================================================================

1. WIRESHARK'TA LOOPBACK ADAPTER'I SEÇİN
   - Capture → Options
   - "Adapter for loopback traffic capture" seçeneğini aktif edin
   - Ya da "Npcap Loopback Adapter" seçin

2. CAPTURE FİLTRELERİ KULLANIN:
   - Port 5000'i dinlemek için: tcp.port == 5000
   - WebSocket mesajları için: websocket
   - HTTP trafiği için: http
   - Hepsi için: tcp.port == 5000 && websocket

3. MESAJLARI GÖRMEK İÇİN:
   - Bir WebSocket paketine çift tıklayın
   - "Line-based text data" veya "WebSocket" satırına bakın
   - "Follow TCP Stream" seçeneğini kullanın

4. ALTERNATİF: TARAYICI GELİŞTİRİCİ ARAÇLARI
   - Chrome'da F12 → Network sekmesi
   - "WS" (WebSocket) filtresi uygulayın
   - Mesajları daha kolay görebilirsiniz

5. BACKEND LOG'LARI
   Backend sunucusu çalışırken terminal'de mesajları görebilirsiniz:
   - Gönderilen mesajları
   - Alınan mesajları
   - Şifreleme/deşifreleme adımlarını
